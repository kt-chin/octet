--Controls--
   w 			- iterate up
   s 			- iterate down
   a 			- switch tree file left
   d 			- switch tree file right
   e 			- increase tree width
   q                    - decrease tree width
   c                    - rotate tree angle right
   z                    - rotate tree angle left
   x 		        - produce flashes of light (on/off)
shift + (w/a/s/d)       - move camera
shift + r		- make tree red
shift + g		- make tree green
shift + b		- make tree blue
shift + y		- make tree yellow
shift + h		- make tree white
 ctrl + r		- make scene red
 ctrl + g		- make scene green
 ctrl + b		- make scene blue
 ctrl + y		- make scene yellow
 ctrl + h		- make scene black
 ctrl + x		- make scene white
 ctrl + c		- make scene reset to default colour.
